# DSA (Data Structures and Algorithms) laboratory works

## Description
The repository contains labs completed during DSA course.

The laboratory assignments are implemented in C language.

## Laboratory Works
| Lab | File | Topic |
| --- | --------------- | --------------------------------------------------------- |
| №1   | LinkedList.c | **Abstract Data Type “Singly Linked List”. Creation, input and output of information, element search, element editing, swapping two elements, sorting, and memory deallocation** |
| №2   | LinkedList.c | **Abstract Data Type “Singly Linked List” (Continuation). Adding a new element, deleting an element, inserting an element, determining the address of the last element, determining the length of the list (number of elements), splitting the list into two, merging two lists, writing element data fields to a file, and reading them from a file** |
| №3   | BinaryTree1.c| **Abstract Data Type “Binary Tree”. Use of Iterative Algorithms** |
| №4   | BinaryTree2.c | **Abstract Data Type “Binary Tree”. Use of Recursive Algorithms** |
| №5   | Sortings.c | **Programming Search and Sorting Algorithms in the C Language** |
